## 线程通信

>线程通信一定是多个线程操作同一个资源才需要进行线程通信
>
>线程通信必须先保证线程安全,否则毫无意义,代码也会报错

~~~java
// 让当前线程进入等待状态,此方法必须有锁对象调用
public void wait(); 

// 唤醒当前锁对象上等待状态的某个线程,此方法必须由锁对象调用
public void notify();

//唤醒当前锁对象上等待状态的全部线程,此方法必须锁对象调用
public void notifyAll();

~~~
